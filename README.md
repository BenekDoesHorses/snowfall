# ❄️ Snowfall

A project made for "Snowfall Ponymeet" event performed online in 2021 by Bronies Silesia.

Building on Windows:
```
git clone git://github.com/noskla/snowfall.git
cd .\snowfall\
.\build.ps1
```
Builing on Linux/FreeBSD:
```
git clone git://github.com/noskla/snowfall.git
cd snowfall
chmod +x build.sh
./build.sh
```
